Pseudo-elementos em CSS

O que são pseudo-elementos?

Os pseudo-elementos (representados por :: em CSS) são extensões dos seletores que mexem diretamente no conteúdo de um elemento através dos seletores genéricos, classes ou IDs. Eles servem para alterar partes de um elemento que o HTML é incapaz de selecionar.

Resumo Seletores Personalizados CSS (até agora)

  • #   id
  • .   class
  • :   pseudo-class
  • ::   pseudo-element
  • >   children
