Grouping Tags

O que são Grouping Tags?

Essencialmente, Grouping Tags são tags usadas para agrupar outras tags e estruturar o site e seu layout.

No HTML4, era muito comum o aninhamento de várias <div> com IDs nomeando suas funções para estruturar o site. Hoje, apesar de não ser considerado errado, tem-se um método mais moderno para realizar essa função: o uso das Grouping Tags Semânticas. Elas não só identificam os elementos com mais clareza, mas serve como um apoio para os sistemas de busca darem mais valor ao seu conteúdo principal e menos aos secundários.

Algumas das principais Grouping Tags Semânticas são:

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <aside>
  • <footer>

Todas essas Grouping Tags são, no final das contas, tratadas como divs visualmente, mas possuem significado dentro do site. Além disso, todas elas podem ser aninhadas uma dentro das outras de maneira indefinida, de forma que não há regra que delimite o uso de uma tag dentro da outra. Fica ao seu critério como queira estruturar seu site!
