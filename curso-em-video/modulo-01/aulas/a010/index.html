Usando Links Externos

Links externos te redireciona para outros sites na internet. Por exemplo, você pode criar um link para o perfil do GitHub do melhor Dev de todos, o Ícaro. Além disso, podemos utilizar um link para redirecionar para o canal Curso em Vídeo no YouTube, que está sendo essencial para este estudo. Estes links são possíveis de serem feitos a partir do uso da tag <a>.

Comportamento dos Links

Para que links externos não substituam a página em que o seu site está aberto, existe dois parâmetros que podem ser atribuídos à tag <a> que fazem com que o site do link seja aberto em uma nova aba do seu navegador. Eles são: target e rel. Atribuindo os seguintes valores para os parâmetros: target:"_blank" e rel="external", conseguimos o comportamento citado.

O parâmetro target tem como objetivo especificar onde que o link será aberto (em que janela ou aba). Já o rel especifica a relação entre o seu site e o link, sendo importante para SEO e segurança.

Links Internos

Links internos são usados quando desejamos redirecionar o usuário para outra página HTML que está hospedada no mesmo servidor.

Usando nofollow e sponsored

Às vezes, marcas podem comprar espaços no seu site através de links. Para isso, o parâmetro rel tem um valor específico: sponsored. Outro valor importante é o nofollow, este serve como forma de assegurar que o conteúdo relevante ou controlado do seu site está unicamente naquela página, e que o link que você colocou não deve ser escaneado pelo algoritmo.

Links Internos para Pastas

Quando usamos links externos, devemos colocar a URL inteira no nosso href. Por outro lado, quando queremos referenciar um arquivo HTML dentro da pasta do nosso index.html, utilizamos apenas o nome do arquivo. Mas e se nosso arquivo estiver dentro de outra pasta? Nesse caso, devemos usar o caminho correto percorrendo a pasta e referenciando o nome do arquivo no nosso href, como está acontecendo neste link para a terceira página do nosso site.
