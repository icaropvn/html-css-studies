Código-Fonte

Exemplo de código em linguagem C: 

            
    #include <stdio.h>

    int main()
    {
        printf("Hello, World!");
        return 0;
    }

Como você pode notar, o texto acima (referente a um código) passou a ser monoespaçado após a utilização da tag <code>.
