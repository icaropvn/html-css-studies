Trabalhando com Formulários, Pt. VI

Novos Inputs

Color

Usado para selecionar uma cor entre todas possíveis.

Range

Usado para definir a quantidade ou nível de algo. O tamanho é personalizável, sendo de 0 a 100 o padrão.

File

Usado para selecionar arquivos e enviá-los. É importante reforçar que para usar esse tipo de input, é necessário usar o método post de envio.
