Novos Elementos

Input tipo email

Inputs do tipo email não fazem validações tão complexas, apenas verificam a existência de um @ no input.

Input tipo tel

Input usado para números de telefone.

Um tópico muito interessante e útil a ser estudado é RegEx em HTML5 (expressões regulares). Essa tecnologia permite manipular dígito a dígito como será a forma do conteúdo do input. Por exemplo: (XX) XXXXX-XXXX para telefones, XX/XX/XXXX para datas, dentre muitos outros.

A Tag fieldset

A tag fieldset serve para agrupar inputs dentro de um mesmo formulário. Podemos usá-la em conjunto com a tag legend para identificar aquele conjunto de inputs.
