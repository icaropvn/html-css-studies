Trabalhando com Formulários, Pt. III

Novos Elementos

Input tipo number

Inputs tipo number aceitam somente números, inclusive colocam setinhas do lado do input para facilitar sua manipulação. Alguns atributos interessantes são: min e max para alterar os valores permitidos, e step para ajustar os intervalos aceitos.

Input tipo month

Inputs desse tipo permitem a escolha de um mês e um ano, a partir de uma caixa formatada para tal. Ao enviar o formulário, tais informações são mandadas da seguinte maneira: ANO-MÊS (ex. 2024-08, 2020-03, 2022-02, etc).

Input tipo date

Input usado para enviar datas específicas. O formato do envio é ANO-MÊS-DIA (ex. 2024-08-25).

Input tipo time

Input usado para enviar horários, sendo possível definir a hora e os minutos. Formato de envio: HORA%3AMINUTOS, sendo o "%3A" o código para o hífen (ex. 19%3A00, 22%3A15).
