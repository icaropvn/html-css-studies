Trabalhando com Formulários, Pt. II

Novos inputs e atributos

Input tipo senha

Inputs do tipo senha trocam a visualização dos caracteres do input por bolinhas, para escondê-la.

Input tipo reset

Inputs do tipo reset limpam todo input dentro do formulário em questão.

Atributo required

Para impedir que o formulário seja enviado sem que os campos estejam preenchidos, tornando-os obrigatórios, podemos usar o atributo "required" dentro dos inputs que possuem tal característica.

Atributos minlength e maxlength

Para atribuir quantidades mínimas e máximas de caracteres dentro dos inputs, basta utilizar os atributos "minlength" e "maxlength".

Atributo size

Podemos usar o atributo "size" para expandir ou diminuir o tamanho da caixinha do input, consequentemente limitando a quantidade de dados que aparecem de uma vez na caixa.

Atributo placeholder

Placeholders são textos auxiliares que aparecem dentro dos inputs para facilitar a percepção do usuário no que deve ser escrito naquele input.

Atributo autocomplete em forms e inputs

Para facilitar a vida do usuário, o atributo "autocomplete" pode ser colocado dentro da tag form para que os inputs sugiram credenciais anteriormente usadas e salvas no cache do navegador. No entanto, este atributo também pode ser usado em inputs para especificar qual é o tipo daquele input e qual sugestão o navegador deve fazer.
