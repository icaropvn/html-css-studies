Trabalhando com Formulários, Pt. VII

Novas Tags

Select

Inputs select são usados para fornecer ao usuário opções já prontas das quais podem ser escolhidas mas não podem ser modificadas. Ao usar a tag select, você deve utilizar a tag option para cada opção que você desejar adicionar. Além disso, essas opções podem ser agrupadas com optgroup, e esses grupos podem ser identificados com o atributo label. Por fim, para fixar uma opção específica e torná-la padrão, basta utilizar o atributo selected na option.

Datalist

Datalists são uma alternativa aos selects para ocasiões em que você deseja dar sugestões de opções ao usuário, mas ao mesmo tempo permitir a inserção manual de outras opções. Para utilizá-la basta aplicar a tag datalist e cada uma das sugestões com as tags option. No entanto, essa tag não funcionará sozinha, ela deve ser linkada a um outro input através do seu ID. Para isso, basta usar o atributo list do input, dando como valor o ID do datalist.

Textarea

Textareas são usadas para transferência de textos maiores. Para utilizá-la, basta usar a tag textarea. Alguns atributos interessantes para sua manipulação são: cols para definir a quantidade de colunas mostradas na caixa, e rows para as linhas. É importante destacar o uso do método de envio POST do formulário quando usar essa tag, uma vez que os textos podem passar de 3KB.
