Pseudo-classes em CSS

Representação

A representação das pseudo-classes em CSS é feita através do : (dois pontos).

O que são pseudo-classes?

Pseudo-classes devem estar relacionadas a uma classe ou elemento, obrigatoriamente. Elas são, nada mais que, ESTADOS de um determinado elemento. (ativo, marcado, habilitado, etc...). Uma pseudo-classe famosa é a :hover, que significa: "ao passar o mouse por cima... faça tal coisa".
