Utilizando Classes (class) em CSS

O que são classes?

As class em CSS são usadas para contornar uma limitação dos identificadores (ids): a atribuição de um mesmo estilo para vários elementos.

Formas de atribuir classes

Você pode usar as classes em CSS em duas formas sintáticas:

  • Apenas colocando .{classe}
  • Atribuindo o nome do elemento antes: h2.{classe}

A diferença está no fato de, ao atribuir o nome do elemento, as classes só serão válidas para aquele tipo de elemento especificamente, enquanto a outra forma abrange qualquer elemento que tiver a classe selecionada.

Hierarquia de Seletores CSS

A hierarquia dos seletores CSS segue a seguinte ordem: genéricos -> class -> id.

Isso influencia no fato de estilos de class sobreporem estilos genéricos, e, por sua vez, ids sobreporem class.

Observações

  • Elementos individuais podem receber MAIS DE UMA CLASSE ao mesmo tempo! Para isso, basta atribuir, dentro do atributo class, os nomes das classes das quais você deseja que aquele elemento tenha, separados por espaço, apenas.
  • Não dê nome para as suas classes com base nos estilos que ela vão receber. Priorize a FUNÇÃO que aquele elemento terá.
