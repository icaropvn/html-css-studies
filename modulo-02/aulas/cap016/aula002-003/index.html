Modelo de Caixas (Box Model)

Ver exemplo

O que é o Box Model?

Box Model é o nome dado ao sistema usado pelo HTML para tratar seus elementos. Internamente, todos os elementos são caixas. Por isso, ao usar um seletor CSS para modificar determinado elemento, na verdade estamos modificando sua box por inteiro. Isso é possível de ser visto ao utilizar a propriedade "background-color", que pinta toda a área da box, não somente o texto, por exemplo.

Esse sistema permite o aninhamento de boxes (colocar uma caixa dentro da outra), sendo extremamente útil para definir alinhamentos específicos.

Anatomia das Boxes

Por ser um sistema padrão, toda caixa, consequentemente todo elemento, possui algumas características padrões que sempre estarão presentes. Elas são:

  • Border - delimita as bordas ou limites da caixa.
  • Padding - traduz-se como acolchoamento ou preenchimento. É o espaço definido entre o conteúdo da box e sua Border (borda). Pode ser chamado de "espaçamento interno".
  • Margin - é a margem da caixa. Se trata do "espaçamento externo" à caixa, sendo o espaço entre a caixa atual e as outras caixas em volta, além da Border.
  • Outline - traduz-se como contorno ou traçado, e se trata de uma "borda extra" que se apresenta externamente à Border, gerando um novo destaque em forma de contorno.

Shorthands Úteis

Podemos usar shorthands para simplificar a aplicação de estilos dos elementos vistos acima. Elas são:

  • border: {width} {style} {color};
  • padding: {top} {right} {bottom} {left}; (sentido horário)
    • Se as 4 medidas forem iguais, basta usar: padding: {medida};
    • Usando 2 medidas, você muda as medidas de forma vertical e horizontal, respectivamente, ou seja:

      padding: {top e bottom} {left e right};

  • margin: {top} {right} {bottom} {left};
  • outline: {width} {style} {color};

Tipos de Box

Boxes podem estar inseridas em duas categorias distintas, cada uma com suas características específicas: a categoria box-level ou inline-level.

Boxes do tipo box-level são definidas como sempre possuindo uma linha única para si. A atribuição de elementos com esse tipo de caixa no seu documento gera uma quebra de linha antes e depois do elemento. Além disso, sua largura (width) sempre será total em relação ao body do documento. Um grande exemplo desse tipo de box é a famosa <div>.

Outros Exemplos de Box-Level

  • <h1> - <h6>
  • <p>
  • <main>
  • <header>
  • <nav>
  • <article>
  • <aside>
  • <footer>
  • <form>
  • <video>

As do tipo inline-level, por outro lado, são exatamente o contrário da anterior. Sua atribuição no documento não gera quebras linha, sendo mostrada sequencialmente ao conteúdo. Além disso, também não necessariamente ocupa todo o width da página, apenas o necessário para mostrar o conteúdo. Um grande exemplo desse tipo é o <span>.

Outros Exemplos de Inline-Level

  • <a>
  • <code>
  • <small>
  • <strong>
  • <em>
  • <sup> / <sub>
  • <label>
  • <button>
  • <input>
  • <select>
