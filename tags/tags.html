TAGs em HTML

Em HTML5, cada elemento usado como conteúdo da sua página é definido por uma TAG, estas que podem possuir ou não fechamentos. Cada TAG é usada para criar um elemento específico, e podem ou não possuir nomes autoexplicativos. Por isso, use a lista abaixo para descobrir a TAG de cada elemento listado.
