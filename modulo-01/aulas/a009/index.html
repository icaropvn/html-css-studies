Trabalhando com Listas

Listas Ordenadas (ordered lists)

Listas Ordenadas são aquelas nas quais a ordem dos itens é estrita, ou seja, significam algo. Nesse tipo de listas, caso um item mude de lugar, seu significado também mudará. A tag usada para obter esse tipo de lista é a <ol>.

Exemplos de OLs: lista de passo-a-passo para fazer algo, listas de vencedores de uma corrida por ordem de chegada, etc.

Exemplo de uso da tag <ol>:

  1. Acordar
  2. Tomar café
  3. Escovar dentes
  4. Faculdade
  5. Almoçar
  6. Trabalho
  7. Casa
  8. Jantar
  9. Dormir

Observações:

  • Ao adicionar um item no meio da lista, ela se reordenará automaticamente;
  • Em HTML5, o uso do fechamento da tag <li> (</li>) se tornou opcional;
  • Há um parâmetro chamado type que pode ser atribuído à <ol> para mudar o estilo da listagem (1, A, a, I, i);
  • Há um parâmetro chamado start, que pode ser atribuído para mudar o valor inicial da listagem.

Listas Não Ordenadas (unordered lists)

Listas Não Ordenadas são aquelas nas quais a ordem não faz diferença. Nesse tipo, a listagem é genérica, usando símbolos idênticos para todos os itens, não gerando uma sequência como no tipo anterior (1, 2, 3, 4...). Para gerar uma lista não ordenada (unordered list) usa-se a tag <ul>.

Exemplo de uso da tag <ul>:

  • Pão
  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão

Observações:

  • O parâmetro type pode ser usado para mudar o estilo do símbolo da listagem (disc, circle, square);

Listas de Definição (Definition Lists)

Listas de Definição são como dicionários do Python. Para cada item da lista, há uma definição para tal. Os termos (ou itens) são identificados com a tag <dt>, já as definições são identificadas com a tag <dd>.

Exemplo:

HTML
Linguagem de marcação para a criação do conteúdo de um site.
CSS
Linguagem de marcação para a criação do design de um site.
JavaScript
Linguagem de programação para a criação de interatividade de um site.

Observações:

  • Os mecanismos de busca dão prioridade para sites que possuem listas de definição quando o usuário necessita da definição de algum termo.
